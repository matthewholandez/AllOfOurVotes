## Architecture

- Backend (`/backend`): Data scraping, transformation and storage. FastAPI and PostgreSQL.

- Frontend (`/frontend`): Data visualization; pretty much everything you see on the [website.](https://allofourvotes.org) Svelte/SvelteKit.
